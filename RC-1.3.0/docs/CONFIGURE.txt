1 - TEMPLATE CONTEXT PROCESSORS (IMPORTANT):
	Add this context processors for custom conf of real_estate_app 'real_estate_app.context_processors.custom_processor'

2 - TEMPLATES REAL ESTATE APP
	a) Set on server media the media dir location of real_estate_app.
	   Exemple: 'Alias /media-real/   <localization of apps>/real_estate_app/media/'

	b) You can custom this localization of media to server, just set on settings project the val REAL_APP_MEDIA_PREFIX.
	   Default: '/media-real/'.
	   Exemple:
	   REAL_APP_MEDIA_PREFIX='/real/'

3 - CUSTOM SITE NAME
	a) Set on settings of project the val REAL_ESTATE_SITE_NAME
	   Default: ''
	   Exemple: REAL_ESTATE_SITE_NAME='Test of site name'

4 - CUSTOM A MAX PROPETY SHOWING ON LIST
	a) Set on settings of project the val PROPERTY_NUM_LATEST
	   Default: 25
	   Exemple: PROPERTY_NUM_LATEST=10

5 - CUSTOM A IMAGES SIZE OF DESTAQUE
	a) Set on settings of project the val REAL_ESTATE_IMAGES_SIZE
	   Default: (626,286)
	   Exemple: REAL_ESTATE_IMAGES_SIZE=(900,300)
